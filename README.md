# 🚀 Kotlin + Ktor Portfolio

Premium, modern, dark-mode portfolio backend built with Kotlin + Ktor — server-rendered with FreeMarker, JSON API included.

## Features
- ⚡ Ktor + Netty server
- 🎨 FreeMarker templating, dark mode default
- 🔌 REST API (`/api/projects`, `/api/skills`)
- 📱 Fully responsive
- 🐳 Docker-ready

## Tech Stack
Kotlin · Ktor · FreeMarker · kotlinx.serialization

## Run Locally
\`\`\`bash
./gradlew run
\`\`\`
